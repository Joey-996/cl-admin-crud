# cl-admin-crud

## Install

#### npm

```shell
npm i cl-admin-crud
```

#### yarn

```shell
yarn add cl-admin-crud
```

#### Browser

```html
<script src="https://cdn.jsdelivr.net/npm/cl-admin-crud@1.3.0/dist/cl-admin-crud.min.js"></script>
```
